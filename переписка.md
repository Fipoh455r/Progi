**Команды:**
```bash

localai serve        # веб на http://localhost:8080
localai models       # список моделей
localai pull <model> # скачать модель
```

## Сессия 4 — 26 июня 2025

### Запрос 5
**Пользователь:**
> Создать файлы-шпаргалки которые сжимают контекст проекта с ~15 000 токенов до ~600, и продолжить улучшение по плану (v2.2 — авторизация).

**Агент:**
Создан `CONTEXT.md` — шпаргалка ~400 токенов: файловая карта, API, версии, шаблоны.

Реализована v2.2 Авторизация (stdlib only):

| Файл | Изменения |
|------|-----------|
| `go/auth.go` | Новый: User/UserStore (JSON), HashPassword (PBKDF2-HMAC-SHA256 100k итераций), VerifyPassword, GenerateJWT/ValidateJWT (HS256), RateLimiter (per-IP скользящее окно), RequireAuth/RequireAdmin middleware |
| `go/server.go` | Переписан v2.2: auth маршруты (/api/auth/setup, /login, /me, /register, /users), все API обёрнуты в RequireAuth, rate limiting для login |
| `go/static/index.html` | Экран входа + первой настройки, JWT в localStorage, authFetch() на всех запросах, logout, admin-панель создания пользователей |

**Маршруты авторизации:**
```
POST /api/auth/setup    — первый admin (только если 0 юзеров)
POST /api/auth/login    → {token, username, role}
GET  /api/auth/status   → {needs_setup: bool}
GET  /api/auth/me       — текущий пользователь (Bearer)
POST /api/auth/register — создать юзера (admin only)
DELETE /api/auth/users/{id} — удалить юзера (admin only)
```
