## ТЕКУЩЕЕ СОСТОЯНИЕ (26 июня 2025 — сессия 7)

### Файлы проекта

```
go/
  main.go        CLI: chat|serve|models|pull|config  флаги: -ollama -model -port -data -config
  ollama.go      ChatStream / ChatStreamWithTemp / chatStreamImpl / ListModels / PullModel
  chat.go        runChat() — цветной терминал с командами /help /clear /model
  storage.go     Storage{Load,Save,Delete,List,GetOrCreate,AppendAndSave} — JSON-файлы
  compress.go    CompressHistory() TrimToTokenBudget() EstimateTokens() — экономия токенов
  tools.go       AllTools: calculator|datetime|web_search|read_file|write_file|http_get
  agent.go       RunAgent() ReAct-цикл: Thought→ToolCall→ToolResult→Answer (макс 8 шагов)
  rag.go         RAG{AddDocument,Search,ListDocs,DeleteDoc} + embedChunks (batch parallel)
  server.go      runServer() v3.1: chat+agent+RAG+compress+OpenAI+auth+graceful+PDF
  openai.go      /v1/models + /v1/chat/completions (stream+sync)
  auth.go        v2.2: User/UserStore + HashPassword/VerifyPassword + JWT HS256
                 + RateLimiter (per-IP) + RequireAuth/RequireAdmin middleware
  audio.go       v2.3: WhisperClient(Transcribe) + PiperTTS(Synthesize)
                 + registerAudioRoutes: /api/audio/transcriptions, /speech, /status
  balancer.go    v3.0: OllamaBalancer{nodes,counter} Round-robin + circuit breaker
  metrics.go     v3.0: Metrics{counters,histograms} Prometheus text 0.0.4
  config.go      v3.1: AppConfig + LoadConfig + MergeEnv + WriteExample (плоский YAML)
  os_exec.go     v3.1: runCommand + runCommandExists + lookupEnvBool (внешние команды)
  tools_test.go  v3.1: 14 тестов (evalExpr, read_file, write_file, RunTool)
  compress_test.go v3.1: 7 тестов (EstimateTokens, TrimToTokenBudget)
  config_test.go v3.1: 9 тестов (LoadConfig, WriteExample, parseBool, MergeEnv)
  static/index.html  SPA v2.3: login/setup + JWT + 🎤 кнопка + MediaRecorder STT
  Dockerfile     multi-stage golang:1.21→alpine:3.19 ~15MB VOLUME /app/data
  go.mod         stdlib only, go 1.21
CONTEXT.md      Шпаргалка ~600 токенов — весь контекст проекта для LLM
```
