## ПРАВИЛА КОДИРОВАНИЯ

```
1. stdlib only — никаких go get, никаких зависимостей
2. go build && go test перед любым коммитом
3. Новый конфиг → config.go (AppConfig + LoadConfig + MergeEnv)
4. Новый инструмент агента → tools.go AllTools["name"] = &ToolDef{...}
5. Новый API-маршрут → server.go mux.Handle("/api/...", protected(handler))
6. Версия → main.go appVersion
```
