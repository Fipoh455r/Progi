## ПРАВИЛА ЭКОНОМИИ ТОКЕНОВ

### Чтение кода
- Читать только нужный файл, не весь проект
- `grep -n "pattern" file.go` вместо полного чтения
- Параллельно читать независимые файлы (множественные read calls)
- `git diff HEAD~1 go/` — только изменения с последнего коммита

### Написание кода
- `filesystem__edit` (patch) для изменений в существующих файлах
- `filesystem__write` только для новых файлов
- Не дублировать код — ссылаться на существующие функции
- Один файл = одна ответственность

### Валидация
- `go build -o /dev/null ./...` — достаточно для проверки компиляции
- `go vet ./...` только при подозрении на семантические ошибки
- Не запускать полные тесты если нет тест-файлов

### Ответы
- Не пересказывать TODO-список
- Ошибки: тип + кол-во → исправить → показать OK
- Успех: одна строка

## ШАБЛОНЫ ЧАСТЫХ ЗАДАЧ

### Добавить новый API эндпоинт
```
1. server.go: mux.HandleFunc("/api/новый", handler)
2. static/index.html: fetch('/api/новый')
3. go build -o /dev/null ./...
```

### Добавить инструмент агента (v2.0)
```
1. go/tools.go: func toolNewTool(args) (string, error)
2. go/agent.go: зарегистрировать в toolMap
3. Обновить системный промпт в chat.go
```

### Добавить поле в Session
```
1. storage.go: добавить поле в SessionSettings или Session
2. server.go: добавить в PATCH-обработчик
3. static/index.html: добавить в settings panel
```

### Пересборка и деплой
```bash
cd go && go build -ldflags="-w -s" -o localai .
# Docker:
docker compose -f docker-compose.goapp.yml up --build -d
```

## OLLAMA API (краткая справка)

```
POST /api/chat    {model, messages:[{role,content}], stream:true, options:{temperature}}
  → stream: {"message":{"role":"assistant","content":"токен"},"done":false}
  → final:  {"done":true,"eval_count":N}

GET  /api/tags    → {"models":[{"name","size","modified_at"}]}
POST /api/pull    {name:"model:tag", stream:true} → {status, total, completed}
POST /api/embeddings {model, prompt} → {"embedding":[...float64]}  ← нужен для RAG
```

## ЧАСТЫЕ ОШИБКИ И РЕШЕНИЯ

| Ошибка | Решение |
|--------|---------|
| duplicate function | Переименовать одну из версий |
| undefined: runXxx | Добавить stub или создать файл |
| go:embed no files | Создать файл до go build |
| SSE не стримится | Проверить X-Accel-Buffering: no |
| session not saved | Проверить dataDir доступен для записи |
