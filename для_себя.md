## ЗАМЕТКИ ДЛЯ СЛЕДУЮЩЕЙ СЕССИИ

1. Начинать с `git pull origin main` — получить актуальный код
2. Проверить `go vet ./...` перед правками
3. Бинарник собирается: `cd go && go build -ldflags="-w -s" -o localai .`
4. Тестировать веб-интерфейс: `OLLAMA_URL=http://... ./localai serve`
5. Токен GitHub сохранён в Pulumi Cloud (запросить у пользователя если истёк)
6. Репозиторий: `https://github.com/Fipoh455r/Progi`
7. Ветки создавать как `neo/feature-name-xxxxx`
